Implementation requirements:

1. **Imports**: Terminal from `@xterm/xterm`, FitAddon from `@xterm/addon-fit`, WebglAddon from `@xterm/addon-webgl`, CanvasAddon from `@xterm/addon-canvas`. Import types `ITerminalOptions` and `ITheme` from `@xterm/xterm`.

2. **Cyberpunk ITheme** (per RESEARCH.md Section 2):
```typescript
const cyberpunkTheme: ITheme = {
  background: '#0a0a0a',

  brightCyan: '#40e0ff',
  brightWhite: '#ffffff',
};
```

3. **Terminal options** (MUST set `allowTransparency: true` for CRT overlays):
```typescript
const term = new Terminal({
  cursorBlink: true,
  cursorStyle: 'block',
  fontFamily: "'JetBrains Mono', 'Consolas', 'Monaco', monospace",
  fontSize: 14,
  lineHeight: 1.2,
  allowTransparency: true,
  convertEol: true,
  scrollback: 500,
  theme: cyberpunkTheme,
});
```

4. **Load addons**: FitAddon first, then try WebglAddon with CanvasAddon fallback (per RESEARCH.md):
```typescript
const fitAddon = new FitAddon();
term.loadAddon(fitAddon);
try {
  const webglAddon = new WebglAddon();
  webglAddon.onContextLoss(() => {
    webglAddon.dispose();
    term.loadAddon(new CanvasAddon());
  });
  term.loadAddon(webglAddon);
} catch {
  term.loadAddon(new CanvasAddon());
}
```

5. **Open terminal**: `term.open(containerRef.current)` then `fitAddon.fit()`.

6. **Input handling** — use `onData` (NOT `onKey`, per RESEARCH.md Section 1 pitfalls):
- Code 13 (Enter): write `\r\n`, call `onCommand(currentInput.trim())` if non-empty, reset buffer.
- Code 127 or 8 (Backspace): if buffer has chars, slice last char, write `\b \b`.
- Code 27 (Escape): ignore for now (arrow key history handled in Plan 03).
- Otherwise: append to buffer, write the character.

7. **ResizeObserver**: Watch container, debounce with `requestAnimationFrame`, call `fitAddon.fit()`.

8. **Cleanup** (CRITICAL for React 18 Strict Mode, per RESEARCH.md):
```typescript
return () => {
  resizeObserver.disconnect();
  if (terminalRef.current) {
    terminalRef.current.dispose();
    terminalRef.current = null;
  }
};
```

9. **Render**: `<div ref={containerRef} className={className} style={{ width: '100%', height: '100%' }} />`

10. **Boot message**: After `term.open()`, write a welcome message:
```typescript
term.writeln('\x1b[1;32mAI Idle Hacker Terminal v2.1.0\x1b[0m');
term.writeln('\x1b[1;36mType "help" for available commands\x1b[0m');
term.write('\x1b[1;32m$\x1b[0m ');
```

Do NOT modify TerminalInterface.tsx yet — that will be replaced in Plan 03.</action>
